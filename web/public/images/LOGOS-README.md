# Logos de pauvepe

Este directorio contiene todos los logos de la marca pauvepe en formato SVG.

## Archivos Disponibles

### 1. `logo-horizontal-blue.svg`
**Uso:** Header del sitio web (Navbar)
- Logo horizontal con texto "pauvepe"
- Color: Azul (#0062FF)
- Dimensiones: 400x100px
- **Actualmente en uso:** Header y Footer

### 2. `logo-horizontal-white.svg`
**Uso:** Fondos oscuros o imágenes
- Logo horizontal con texto "pauvepe"
- Color: Blanco (#FFFFFF)
- Dimensiones: 400x100px
- **Sugerencia:** Usar en secciones con fondo azul oscuro o en el footer si cambias el fondo a oscuro

### 3. `logo-vertical-blue.svg`
**Uso:** Perfiles de redes sociales o firmas
- Logo vertical con icono arriba y texto abajo
- Color: Azul (#0062FF)
- Dimensiones: 300x200px
- **Sugerencia:** Ideal para Instagram, LinkedIn, etc.

### 4. `logo-icon.svg`
**Uso:** Favicon o imagen de perfil circular
- Solo el isotipo (icono sin texto)
- Color: Azul (#0062FF)
- Dimensiones: 100x100px
- **Sugerencia:** Convertir a favicon.ico para el navegador

## Colores de Marca

- **Azul Principal:** `#0062FF`
- **Blanco:** `#FFFFFF`

## Tipografía

Los logos utilizan la fuente **Montserrat Black (900)** importada desde Google Fonts.

## Notas de Implementación

- Todos los logos están en formato SVG vectorial, escalables sin pérdida de calidad
- Los archivos incluyen la fuente Montserrat mediante `@import` de Google Fonts
- El logo horizontal azul está actualmente implementado en el Header y Footer del sitio web
